# Shutdown a Debian Guest

<br>
## RUN: Shutdown from console/tty1 (ALT-F1)

* login as user: root
* enter the root password
* execute "shutdown"

```
shutdown -h now

```

<img src="pictures/WK-OSP-18.jpg"  width="960">

<br>
## PR RUN: poweroff from console/tty1 (ALT-F1)

* login as user: root
* enter the root password
* execute "poweroff"


```
poweroff

```
