## clone the Operating Systems class repository

```

[ -d $HOME/git/ ] || { echo "ERROR: no $HOME/git/ directory!"; exit 1; }
cd $HOME/git/
git clone git@github.com:os2xx/os.git
ls -aF
cd os
git status
ls -aF
```
